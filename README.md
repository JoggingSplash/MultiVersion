# MultiVersion ext-encoding

A multiversion implementation originally developed by [Flonja](https://github.com/Flonja) for [PocketMine-MP](https://github.com/pmmp/PocketMine-MP).

## Disclaimer

Packet translations may be incomplete, incorrect (or wrong translated) across certain versions.
As a result, unexpected behavior or incompatibilities may occur.

Contributions are welcome!  
If you find issues or have improvements, feel free to open a pull request or submit a report.

## Running Versions
These are some of the versions tested and working at the time we wrote this
